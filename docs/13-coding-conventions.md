- USE PSR-2

## Confiuration

- MUST NOT contain secrects

## Database (MySQL)

- camelCase_id

## PHP

### Classes

- MUST USE English names

### Variables

- camelCase

### Yii 

- SHOULD USE `Yii::info()` or `Yii::trace()`, NOT `Yii::getLogger->(..., ..., ...)`

### Giiant (Backend CRUD)

- SHOULD contain `Id` columns
